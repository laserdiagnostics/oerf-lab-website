# 《加入 OERF：从问题开始》漫画分镜（审稿修订版）

**角色参考**：`characters/characters.png`（人物设定通过后再生成）  
**网站用途**：主页“加入我们 / Join Us”栏目  
**展示方式**：桌面端 2×2，移动端单列  
**文字策略**：成图不放姓名、中文对白、邮箱或长标签；标题与对白作为图片外的网页 HTML，并置于对应图片上方、同一卡片内  
**独立封面**：不生成，栏目标题由网页承担

## 连续性规则

- 固定人物只有申请者林澄和博士生师姐周岚；姓名仅供制作沟通，不出现在图片中。
- 林澄始终保留海军蓝外套或可见的海军蓝内层、细边方形眼镜和暖米色背包。
- 第二页夜场是唯一服装亮度例外：林澄可脱下海军蓝外套露出奶油白上衣，外套搭在附近椅背，并保留暖色边缘光。
- 周岚始终保留低马尾、暖橙色内搭和白色实验服；非实验场景可脱下实验服搭在椅背，但暖橙色内搭不变。
- 不出现虚构导师、国际教授视频肖像、真实高校徽标和真实期刊名称。
- 光学平台、相机、透镜、计算屏幕和小型反应流装置保持专业可信，但不画复杂型号和不可验证的精密结构。
- 第二页与第四页使用同一类火焰装置：透明圆柱形安全罩、简洁金属灰底座、内部小型受控蓝色火焰。
- 白板、屏幕和记录纸不得出现可读文字或类似文字的笔画；只允许纯图形线条、箭头、曲线、色块和无意义涂改痕迹。
- 每张图只表达一个核心感受；不使用概念符号、路径墙或信息图式堆叠。

## 第 1 页：《第一周，从问题开始》

**文件名**：`01-page-start-with-the-question.png`  
**布局**：3:4 竖版，上方主格约 2/3，下方近景格约 1/3  
**核心信息**：没有相关研究经历，也可以从拆解问题开始。  
**叙事阶段**：坦白 → 被理解 → 开始动手

### 主格（上方）

**场景**：现代高校实验室的白板讨论区，普通工作日上午。  
**镜头**：从白板一侧斜向人物的中景，像偶然捕捉到的真实科研日常。  
**人物与动作**：林澄已经站在实验室里，手持白板笔，在白板上画一个很简单的光束、火焰和问号草图；周岚没有替她画答案，而是用另一支笔把问题拆成三个小步骤。两人并肩站立，关系平等。  
**背景细节**：桌边放着林澄的米色背包和简历文件夹；白板只保留少量无文字线条、箭头和图形，不生成公式乱码。  
**情绪**：林澄略紧张但愿意开口，周岚自然、专注、没有居高临下。

### 近景格（下方）

**场景与动作**：两支不同颜色的白板笔同时落在同一个问题草图旁；林澄的手由迟疑变得稳定。  
**视觉重点**：手、笔、清楚的三步箭头和两个人靠近白板的影子。  
**不得出现**：门外犹豫、开门欢迎、导师形象、复杂学科图标。

**图片外 HTML 对白**：

- “我之前没做过实验。”
- “巧了，我博一也不会。先把问题拆开。”

## 第 2 页：《出问题了，就一起拆》

**文件名**：`02-page-debug-it-together.png`  
**布局**：3:4 竖版，三个清晰分格；上方故障、中部协作、下方夜间重搭  
**核心信息**：执行力和解决问题的能力来自真实行动，失败时有人一起复盘。  
**叙事阶段**：失败 → 排查 → 重搭

### 第 1 格（上方，约 1/3）

**场景**：简洁可信的光学实验台。  
**人物与动作**：林澄看见电脑上原本平稳的曲线突然变成噪声，表情短暂错愕；透明圆柱形安全罩和简洁金属灰底座内的小型受控蓝色火焰保持稳定。  
**画面限制**：屏幕只画曲线与噪声，不画文字、数字或软件界面。

### 第 2 格（中部，约 1/3）

**场景**：同一实验台。  
**人物与动作**：周岚与林澄分别从光路和数据两端检查问题；一人调整带旋钮的小型光学支架，一人拿记录本对照前一次结果。两人专注交流，不摆拍。  
**视觉重点**：两双手在同一问题上协作，桌面整洁，安全护目镜佩戴规范。

### 第 3 格（下方，约 1/3）

**场景**：夜晚的实验室，窗外深蓝，室内仍是温暖顶灯与实验台工作灯。  
**人物与动作**：两人重新搭好简化光路，旁边放着两杯咖啡和一张画满修改痕迹的记录纸；新曲线刚开始恢复稳定。林澄脱下海军蓝外套并搭在椅背，奶油白上衣被可调节实验台工作灯照亮，肩部和短发带一圈暖色边缘光；她露出疲惫但释然的笑。  
**情绪**：不是苦情加班，而是“失败以后还能一起把事情重新做起来”。

**图片外 HTML 对白**：

- “这条信号不对。”
- “别急，我们一起从光路开始查。”

## 第 3 页：《认真科研，也认真生活》

**文件名**：`03-page-research-and-life.png`  
**布局**：3:4 竖版，上方公共区生活主格约 3/5，下方成果小庆祝约 2/5  
**核心信息**：团队既重视高质量研究，也有自然、融洽的日常。  
**叙事阶段**：相处 → 收获 → 看见更大的世界

### 主格（上方）

**场景**：实验室公共区的午餐时间。  
**构图约束**：前景只清晰呈现林澄和周岚。两人隔着小桌面对面坐，形成稳定的双人构图；林澄手边是咖啡，周岚已将筷子平放在餐盒边，用自然伸直的食指点向一张随手画的草图，两人自然笑起来。  
**背景成员**：两三位不具备固定身份的团队成员只出现在较远的公共区，使用浅景深柔和虚化，保留坐姿和交谈轮廓即可，不要求清晰脸部、手部或餐具动作。  
**背景细节**：白板上留着当天没有讨论完的简单草图；周岚的实验服搭在椅背；前景桌上只有两套餐盒、咖啡和一杯奶茶，避免杂物过多。  
**限制**：不得把所有成员画成同一张桌旁的清晰群像；背景成员不对应任何真实团队成员，不设置复杂识别物。

### 成果格（下方）

**场景与动作**：电脑屏幕只显示一张论文缩略图和清楚的绿色接收勾选状态，不出现可读期刊或邮件文字；周岚与林澄并肩看着屏幕同时微笑，两人的双手各自自然放在桌面，不接触、不击掌。林澄把一小块蛋糕放在桌子中间。  
**背景细节**：墙上有一张极简世界地图，几个温和光点由细线连接；只象征合作与发展机会，不画校徽、国旗或教授头像。  
**情绪**：克制、真实的小庆祝，不表现保证录取或既定海外去向。

**图片外 HTML 对白**：

- “论文接收了，今晚加菜。”
- “高质量成果会打开更大的世界。”

## 第 4 页：《把结果讲清楚》

**文件名**：`04-page-show-the-result.png`  
**布局**：3:4 竖版，单一收束主场景，可保留下方少量呼吸空间  
**核心信息**：成长落在可以解释、可以验证的结果上；下一步是主动联系。  
**叙事阶段**：独立表达 → 结果稳定 → 开放邀请

### 全页主场景

**场景**：明亮、整洁的实验室，傍晚暖光。  
**镜头**：中景，镜头约在人物胸口高度并轻微仰视；两位人物全身可见但距离不过远，构图简洁。  
**人物与动作**：林澄站在实验台与屏幕之间，主动向周岚展示一组干净的数据曲线或三维场结果；周岚在侧面认真听。安全装置中的受控蓝色火焰稳定燃烧。远处实验室门敞开，柔和光线进入画面。  
**只保留三个主元素**：林澄讲结果、稳定蓝焰、开放的门。  
**不得出现**：四符号回收、世界地图、路径墙、新学生剪影、多束魔法式彩光、密集团队群像。

**图片外 HTML 对白与行动句**：

- “这次，结果讲清楚了。”
- “带上简历，来聊聊你想解决的问题。”

### 漫画卡片阅读顺序

每张漫画卡片固定采用：页标题 → 两行短对白 → 紧贴的漫画图片 → 可选补充说明。对白在图片上方提供阅读引导，与图片保持紧凑间距；不把文字覆盖在漫画图片上，也不在对白与图片之间插入按钮或其他模块。

### 招生对象

课题组常年招收博士后、致远荣誉博士、申请考核博士、联合培养博士、专硕、学硕、联合培养硕士、研究助理和本科实习生。

### 专业背景

欢迎力学、工程热物理、数学、机械、光学工程、物理以及相关理工科专业的同学联系。上海交通大学机械专业学生如对课题组感兴趣，请在申请考核面试前联系团队。

### 培养与发展

本科实习生如产生高质量研究成果，将有机会获得面向剑桥、牛津、帝国理工、哈佛、MIT 等世界名校的推荐与深造支持。博士生有机会参与与国外知名教授（包括诺贝尔奖得主等）的共同培养与科研合作。

### 我们看重

想象力、执行力、解决问题的能力，以及在逆境中重新推进工作的能力。没有相关研究经历也可以联系。
