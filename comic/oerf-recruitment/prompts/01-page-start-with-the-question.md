# Page 01：第一周，从问题开始

## 视觉风格

专业现代日漫画风，温馨的科研日常，成年人比例约 6.5 至 7 头身。干净流畅的线条、自然线宽变化、稍低饱和的暖色、柔和室内日光、奶油白与暖灰背景。人物表情真实克制，不使用夸张萌系大眼、Q 版比例或宣传海报式摆拍。采用 3:4 竖版漫画页，两个上下分格；两个分格之间只用一条细而克制的白色空隙自然分隔，画面仍保持为一张完整漫画页，不断裂成两张独立卡片。阅读方向从上到下。

## 角色参考

- 林澄：23 岁左右中国青年学生，黑色齐耳短发、细边方形眼镜、海军蓝连帽外套、奶油白上衣、深灰长裤。略紧张但愿意开口。
- 周岚：26 岁左右中国女性博士生，黑色低马尾、暖橙色针织上衣、白色实验服、深色长裤。自然、专注、友善，与林澄保持同一视线高度。

## 分格构图

### 上方主格，约占页面三分之二

现代高校实验室的白板讨论区，普通工作日上午。从白板一侧斜向两人的中景，像偶然捕捉到的真实日常，而不是正对镜头。林澄站在白板前，右手拿深蓝色白板笔，画出极简的光束线、受控小火焰轮廓和一个纯图形问号；她略微皱眉但保持目光交流。周岚并肩站在旁边，用暖橙色白板笔将问题拆成三个清楚的无文字小步骤，只用圆点、箭头和简单线条。两人平等讨论，周岚不替林澄完成全部内容。

白板内容稀疏、清楚，只允许纯图形线条、圆点和箭头；不允许公式、文字、数字、类似汉字或字母的笔画。桌边只放暖米色双肩包和透明简历文件夹，文件页保持空白。背景有少量柔和虚化的实验室轮廓和绿植，不能抢夺人物注意力。

### 下方近景格，约占页面三分之一

近距离特写两只握笔的手同时靠近同一个简洁问题草图：海军蓝袖口对应林澄，白色实验服与暖橙袖口对应周岚。两支不同颜色的白板笔停在三步箭头旁，两个人靠近白板的柔和影子进入画面。手部结构自然完整，五指清楚，不重复、不融合。画面表达“从迟疑到共同开始”，不出现人物正脸。

## 完整生成提示词

生成一张 3:4 竖版、两格上下排布的温馨专业日漫漫画页。上方三分之二为白板讨论的自然中景，下方三分之一为两只手和两支白板笔的协作特写。两格只以细而克制的白色空隙自然分隔，保持一张完整漫画页的整体感。保持林澄与周岚的角色参考一致，成人比例、真实科研气质、柔和暖光、奶油白和暖灰空间、海军蓝与暖橙角色色。情绪是“没有经验也可以从拆问题开始”，不画敲门、开门欢迎或导师训话。

严格禁止任何可读文字，包括标题、对白、对白气泡、旁白框、姓名、公式、字母、数字、软件界面、大学徽标、实验室标牌、邮箱和水印。不得出现任何类似文字的笔画、伪汉字、伪字母或类字符痕迹；白板只用纯图形线条、圆点和箭头。不要复杂学科图标，不要概念符号，不要第三个清晰人物，不要奇怪手指，不要人物融合，不要魔法光效，不要摄影写实风。
