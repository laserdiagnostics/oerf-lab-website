# 内容分析（审稿修订版）

## 修订结论

已采纳 `critique.md` 中全部 P0、P1，以及与成片质量相关的 P2/P3 意见：

- 删除虚构“陈老师”，避免读者把原创角色误认为真实导师。
- 将周岚和方启合并为一名跨实验与计算方向的博士生“周岚”，主要人物从四人减为两人。
- 放弃需要解码的四符号系统，改用真实、直观的科研小场景表现想象力、执行力、解决问题和逆境推进能力。
- 删除“门外犹豫—开门欢迎”、成长路径墙、国际教授视频屏幕和堆叠式大结局。
- 由两张 4:3 横版改为四张 3:4 竖版；桌面端 2×2，移动端单列阅读。
- 增加午餐、咖啡、聊天和论文接收后的轻量庆祝，真实表现团队氛围。
- 漫画图片原则上不生成中文对白、姓名、邮箱或长标签。关键对白与招生信息使用图片外的原生 HTML，不对成图做文字覆盖或修补。
- 第三页群像固定为“前景两名主角清晰、远景团队成员柔和虚化”，降低多人面孔和手部生成错误。
- 第二页夜场让林澄露出奶油白上衣，并用室内暖光勾勒人物轮廓，避免海军蓝服装融入暗背景。
- 每张网页卡片使用“标题与短对白在前、漫画图片紧随其后”的阅读顺序，让读者先获得情境再看画面。

最后一点与审稿意见中的“CSS 覆盖空白气泡”不同：网页文字会放在每张漫画卡片的独立说明区，而不是叠加到图片上。这样既便于中英文切换和无障碍阅读，也符合漫画生成流程中“不得用程序覆盖修复成图文字”的要求。

## 核心信息

OERF Lab 面向不同培养阶段和多种理工科背景长期招生。团队看重想象力、执行力、解决问题和在逆境中继续推进工作的能力；没有相关研究经历不是加入的障碍。

漫画需要让读者在短时间内形成四个感受：

1. **我可以开始**：即使没有实验经验，也可以从拆解一个问题开始。
2. **我不会独自面对失败**：团队成员会一起排查、重搭和复盘。
3. **这里既认真科研，也认真相处**：有高标准，也有真实、轻松的日常。
4. **我知道如何联系**：准备个人简历，可附成绩单，通过邮箱联系团队。

## 目标受众

### 主要受众

- 博士后、致远荣誉博士、申请考核博士和联合培养博士申请者。
- 专硕、学硕、联合培养硕士和研究助理申请者。
- 希望进入实验室开展科研训练的本科实习生。

### 学科背景

- 力学、工程热物理、数学、机械、光学工程、物理及相关理工科学生。
- 上海交通大学机械专业、需要在申请考核面试前联系团队的申请者。

## 视觉叙事原则

| 原则 | 执行方式 |
| --- | --- |
| 一页一个核心感受 | 每页只围绕“开始、协作、氛围、行动”中的一个主题 |
| 场景代替符号 | 用白板草图、排查故障、深夜重搭、论文接收和共同用餐直接表达 |
| 人物少而稳定 | 仅保留申请者和博士生师姐两名固定角色；其他成员只作远景虚化氛围 |

## 角色弧线

- **林澄（内部名，图片中不标姓名）**：第一周坦白自己没有实验经验；学会把问题拆开；经历失败与重搭；最终能够清楚展示结果并主动迈出下一步。
- **周岚（内部名，图片中不标姓名）**：跨实验与计算方向的博士生师姐。她不是“讲课的人”，而是和主角一起画、一起查、一起重搭的同伴。
- **背景团队成员**：只在公共区和小型庆祝场景出现，表现自然互动，不建立独立人物弧线，也不对应真实成员肖像。

## 四页叙事方案

1. **《第一周，从问题开始》**：故事从实验室内部直接切入。林澄已经站在白板前，坦白自己没做过实验；周岚和她一起把问题拆成可做的步骤。
2. **《出问题了，就一起拆》**：实验信号失败，两人检查光路、对照数据，并在夜晚重新搭建。用行动表达执行力、解题能力和逆境推进。
3. **《认真科研，也认真生活》**：林澄与周岚在公共区吃饭、喝咖啡、聊天，其他成员作为远景氛围；随后收到论文接收消息，用克制的小庆祝和世界地图光点表现成果带来的更大机会。
4. **《把结果讲清楚》**：林澄展示一组干净可信的数据，受控蓝焰稳定燃烧，实验室门保持敞开。结尾只保留“结果、火焰、开放的门”三个主元素。

## 图片外的网页对白建议

以下句子作为每张图片上方、同一卡片内的 HTML 引导文案，不生成在漫画图像中：

1. “我之前没做过实验。” / “巧了，我博一也不会。先把问题拆开。”
2. “这条信号不对。” / “别急，我们一起从光路开始查。”
3. “论文接收了，今晚加菜。” / “高质量成果会打开更大的世界。”
4. “这次，结果讲清楚了。” / “带上简历，来聊聊你想解决的问题。”

网页实现时应保留人物称谓与语气，但不必显示内部姓名。推荐卡片顺序为“页标题 → 两行短对白 → 漫画图片 → 可选补充说明”；对白与图片之间不插入按钮、分隔栏或其他内容。移动端不使用图片上的 CSS 文字覆盖，而是通过紧凑间距让对白与图片上半部形成一个连续阅读单元。

## 必须保留为网页可编辑文字

## 发布前表述边界

- 海外深造写作“本科实习生如产生高质量研究成果，将有机会获得面向相关世界名校的推荐与深造支持”，不表达保证录取。
- 国际共同培养写作“博士生有机会参与与国外知名教授（包括诺贝尔奖得主等）的共同培养与科研合作”，不画具体教授肖像。
- 论文接收画面不出现真实期刊名称、大学徽标或可辨识的虚构邮件正文。
- 科学装置以可信的研究氛围为目标，不暗示图中设备是团队某一真实装置的精确复刻。
